6. 🌐 SEMANA 2-3: Geofencing (2-3 horas)

Objetivo

Implementar alertas automáticas basadas en zonas geográficas. El sistema detecta cuándo un empleado entra/sale de una zona y envía alertas automáticamente.

PASO 1: Crear Tabla de Geofences en D1

Duración: 15 minutos

En SQL Console de Cloudflare:

CREATE TABLE geofences ( id INTEGER PRIMARY KEY, nombre TEXT NOT NULL, lat REAL NOT NULL, lon REAL NOT NULL, radio INTEGER NOT NULL, -- radio en metros tipo TEXT, -- 'cliente', 'almacen', 'oficina' activo BOOLEAN DEFAULT 1 ); -- Agregar algunos geofences de ejemplo: INSERT INTO geofences (nombre, lat, lon, radio, tipo) VALUES ('Oficina Central', -12.0456, -77.0289, 100, 'oficina'), ('Cliente A', -12.0512, -77.0267, 100, 'cliente'), ('Cliente B', -12.0389, -77.0445, 100, 'cliente');

Click: Run

PASO 2: Crear Tabla de Alertas

Duración: 10 minutos

CREATE TABLE alertas_geofence ( id INTEGER PRIMARY KEY, user_id TEXT NOT NULL, geofence_id INTEGER NOT NULL, tipo TEXT, -- 'entrada' o 'salida' timestamp DATETIME DEFAULT CURRENT_TIMESTAMP, lat REAL, lon REAL, enviada BOOLEAN DEFAULT 0 );

Click: Run

PASO 3: Lógica de Geofencing en Worker

Duración: 45 minutos

En Antigravity, crea: src/geofencing.js

// Función para calcular distancia entre dos puntos GPS function distancia(lat1, lon1, lat2, lon2) { const R = 6371000; // Radio de la Tierra en metros const dLat = (lat2 - lat1) * Math.PI / 180; const dLon = (lon2 - lon1) * Math.PI / 180; const a = Math.sin(dLat/2) * Math.sin(dLat/2) + Math.cos(lat1 * Math.PI / 180) * Math.cos(lat2 * Math.PI / 180) * Math.sin(dLon/2) * Math.sin(dLon/2); const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a)); return R * c; } // Función principal de geofencing export async function verificarGeofences(userId, lat, lon, db) { const geofences = await db.prepare( 'SELECT * FROM geofences WHERE activo = 1' ).all(); const alertas = []; for (const gf of geofences.results) { const dist = distancia(lat, lon, gf.lat, gf.lon); const dentroDeZona = dist <= gf.radio; // Verificar último estado const ultimaPos = await db.prepare(` SELECT * FROM ubicaciones WHERE user_id = ? ORDER BY timestamp DESC LIMIT 1 `).bind(userId).first(); if (ultimaPos) { const distAnterior = distancia( ultimaPos.latitude, ultimaPos.longitude, gf.lat, gf.lon ); const estabaFuera = distAnterior > gf.radio; // Si estaba fuera y ahora está dentro = ENTRADA if (estabaFuera && dentroDeZona) { alertas.push({ tipo: 'entrada', geofence: gf.nombre, userId: userId }); } // Si estaba dentro y ahora está fuera = SALIDA if (!estabaFuera && !dentroDeZona) { alertas.push({ tipo: 'salida', geofence: gf.nombre, userId: userId }); } } } return alertas; }

Ahora integra esto en tu worker:

import { verificarGeofences } from './geofencing'; export default { async fetch(request, env, ctx) { const { lat, lon, userId } = await request.json(); const db = env.DB; // Guardar ubicación await db.prepare(` INSERT INTO ubicaciones (user_id, latitude, longitude, timestamp) VALUES (?, ?, ?, ?) `).bind(userId, lat, lon, new Date().toISOString()).run(); // Verificar geofences const alertas = await verificarGeofences(userId, lat, lon, db); // Guardar alertas for (const alerta of alertas) { await db.prepare(` INSERT INTO alertas_geofence (user_id, geofence_id, tipo) SELECT ?, id, ? FROM geofences WHERE nombre = ? `).bind(userId, alerta.tipo, alerta.geofence).run(); } return new Response(JSON.stringify({ status: "ok", alertas: alertas })); } };

Deploy nuevamente: wrangler deploy

PASO 4: Integración de Alertas (Telegram/WhatsApp)

Duración: 30 minutos

Opción A: Telegram Bot (Más fácil)

  • 1. Abre Telegram → busca @BotFather
  • 2. /newbot → sigue instrucciones
  • 3. Copias el TOKEN
  • 4. En wrangler.toml: secret: TELEGRAM_TOKEN = [tu_token]

Función para enviar alerta:

async function enviarAlertaTelegram(alerta, env) { const token = env.TELEGRAM_TOKEN; const chatId = env.TELEGRAM_CHAT_ID; const mensaje = ` ⚠️ ALERTA GEOFENCE Usuario: ${alerta.userId} Evento: ${alerta.tipo} en ${alerta.geofence} Hora: ${new Date().toLocaleTimeString()} `; await fetch(`https://api.telegram.org/bot${token}/sendMessage`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ chat_id: chatId, text: mensaje }) }); }

Llama esta función cuando haya alertas en tu Worker
